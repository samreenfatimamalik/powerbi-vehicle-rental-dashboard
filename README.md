# powerbi-vehicle-rental-dashboard
This project is an interactive Power BI Dashboard built to analyze the financial transactions of a vehicle rental company. It provides insights into revenue, cost, profit, customer performance, and business segments, helping stakeholders make data-driven decisions.
Here's a professional README.md you can use for your GitHub repository.



Objectives

Monitor overall financial performance.

Compare revenue and cost across transaction types.

Analyze customer-wise revenue.

Identify the most profitable business segments.

Provide interactive filtering for better analysis.

 Insights

Rental and Sale transactions contribute significantly to overall revenue.

Purchase transactions represent the highest operational cost.
